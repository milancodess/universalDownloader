How to redeploy

  1. Update environment variables or secrets on Vercel as needed.
  2. Run npm install then npm run dev locally to verify.
  3. Push to main (or open a PR) to trigger a fresh Vercel deploy.

Need a direct API link?

Every endpoint uses the same contract: send a url query and receive back download metadata. You can still copy/paste the URL into your own scripts, Postman collections, or production workloads.
